# A股涨停潜力股追踪系统 - 设计理念

## 1. 核心设计理念：专业、敏锐、科技感

我们的目标是打造一个既专业又具有现代科技感的股票分析平台。设计将强调数据的清晰度、实时性和分析的深度，同时通过视觉元素传达出“精准捕捉市场机会”的敏锐感。

**Design Movement**: **Cyber-Financial Tech (赛博金融科技)**

**Core Principles**:
1.  **Data-First Clarity**: 数据可视化是核心，图表和数字必须清晰易读，避免过度装饰干扰信息获取。
2.  **Dynamic Pulse**: 通过动态效果（如呼吸灯、流光）传达市场的实时跳动和活力。
3.  **Precision & Trust**: 使用精细的线条、网格和深色背景，营造专业、严谨和值得信赖的氛围。
4.  **High Contrast**: 利用高对比度色彩突出关键信息（如涨停、买入点），引导用户视线。

**Color Philosophy**:
-   **Background**: 深邃的午夜蓝 (Midnight Blue) 或 炭黑 (Charcoal Black)，营造沉稳的金融科技感，减少长时间盯盘的视觉疲劳。
-   **Primary Accent**: 霓虹红 (Neon Red) 代表上涨/涨停，这是A股市场的核心色彩，需高亮且充满活力。
-   **Secondary Accent**: 荧光绿 (Fluorescent Green) 代表下跌/支撑位，与红色形成鲜明对比。
-   **Neutral**: 银灰色 (Silver Grey) 用于文字和辅助图形，保持冷静和客观。
-   **Highlight**: 金色 (Gold) 用于标记核心推荐或高潜力股，象征财富和价值。

**Layout Paradigm**:
-   **Dashboard-Centric**: 采用仪表盘式布局，关键信息一目了然。
-   **Modular Grid**: 使用模块化网格系统，灵活展示不同维度的分析数据（如热点板块、个股详情、资金流向）。
-   **Asymmetric Balance**: 避免绝对对称，通过左侧导航/核心数据与右侧详细分析/图表的非对称平衡，增加视觉张力。

**Signature Elements**:
-   **Data Streams**: 背景中隐约可见的数据流或代码雨，增强科技感。
-   **Glowing Edges**: 卡片和图表边缘带有微弱的发光效果，增加层次感。
-   **Tech Lines**: 细致的科技线条装饰，连接不同模块，象征信息的流动。

**Interaction Philosophy**:
-   **Hover-Reveal**: 鼠标悬停时展示更多细节数据，保持界面整洁。
-   **Real-time Updates**: 数据变动时有微小的闪烁或颜色变化，即时反馈市场动态。
-   **Smooth Transitions**: 页面切换和模块展开采用流畅的过渡动画，体现系统的高效。

**Animation**:
-   **Entrance**: 模块按优先级依次滑入或淡入。
-   **Chart Growth**: 图表加载时有动态生长效果。
-   **Pulse**: 重点推荐股票有轻微的呼吸效果。

**Typography System**:
-   **Headings**: 使用现代无衬线字体 (如 Roboto, Inter)，加粗，体现力量感。
-   **Numbers**: 使用等宽字体 (如 JetBrains Mono, Fira Code)，确保数字对齐和易读性。
-   **Body**: 清晰易读的无衬线字体，字重适中。

-   **Background**: 纯白 (Pure White) 或 极浅灰 (Off-White)，营造干净、通透的空间感。
-   **Primary**: 深黑 (Jet Black) 用于文字和主要图表线条，极致对比。
-   **Accent**: 这里的红绿色彩饱和度较低，更偏向于莫兰迪色系，避免视觉刺激，保持冷静。
-   **Highlight**: 使用单一的亮色（如克莱因蓝）作为交互和重点提示。

**Layout Paradigm**:
-   **Single Column / Split Screen**: 采用单栏流式布局或左右分屏，阅读体验类似深度研报。

-   **Thin Lines**: 极细的分割线。

## 3. 选定方案：Cyber-Financial Tech (赛博金融科技)

我们将采用 **Cyber-Financial Tech** 风格。这种风格最能契合“实时追踪”、“涨停潜力”和“精准分析”的主题。深色背景适合长时间查看，高对比度的红绿配色符合A股股民的习惯，科技感元素则增强了系统的专业性和先进性。

**具体执行：**
-   **背景**：深色渐变背景，叠加微弱的网格纹理。
-   **卡片**：半透明磨砂玻璃效果 (Glassmorphism)，深色底，带微弱边框发光。
-   **图表**：使用 ECharts 或 Recharts，定制红绿配色，线条平滑，带阴影区域。
-   **字体**：标题使用 Inter (Bold)，数字使用 JetBrains Mono。
-   **动效**：关键数据变化时有数字滚动效果，涨停股列表有流光扫过。
